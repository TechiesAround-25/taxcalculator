Supported Columns

Required: `First Name`, `Last Name`, `Frequency`, `Pension Withdrawl`, `TaxFree Portion`, `Tax Code`, `Cummulative`, `YTD Gross`, `YTD Tax`, `Current Period No`

Also supported: `Actual YTD`, `Actual YTD Tax`, `Tax`, `Current Tax`, `Gross`, `Pension Withdrawal`, `TaxFree Mode`, `SSAS`, `Only Income`

Recognised frequency values: `Monthly`, `Weekly`, `Annual`.

`Cummulative` accepts values like `Yes`, `No`, `True`, `False`, `1`, `0`.

If a CSV uses `Actual YTD` and `Actual YTD Tax`, the importer treats them as cumulative totals including the current row and converts them back to prior YTD before calling the API.

Each row is sent to the API sequentially so you can trace failures row by row instead of masking them in a bulk request.
